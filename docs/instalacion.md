# Instalación
La informaciónd este apartado es para descargar la documentación y no el projecto real.

## Requisitos
Para visualizar o editar:

* **Navegador web** moderno (Chrome, Firefox, Edge, etc.).
* **Acceso a GitHub** para ver el repositorio.
* **Visual Studio Code**: Recomendado para editar los archivos `.md`.

## Pasos de instalación
1. **Clonar el repositorio**: Descarga una copia local del proyecto desde GitHub.
2. **Abrir en el editor**: Importa la carpeta en Visual Studio Code.
3. **Instalar dependencias**: Configura el servidor local si necesitas previsualización avanzada.
4. **Previsualizar**: Usa la función "Open Preview" de VS Code (Ctrl+Shift+V) o levanta el servidor local.
